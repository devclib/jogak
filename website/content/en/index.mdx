# Jogak

> Pronounced "**조각**" — Korean for **"piece"**. Small pieces gathered into a whole.

**Jogak** is a lightweight, fast component showcase. A drop-in alternative to Storybook with the same workflow (`dev` / `build` / static deploy) but a leaner stack — Single Vite, single iframe scope for user components, ts-morph for type extraction, Preact for Web Components.

> **Coming from Storybook?** [Migration guide](/en/docs/migration-from-storybook) · [Full comparison + addon matrix](/en/docs/comparison)

> Status: **`1.0.0`** — stable release. Public API is frozen; future changes follow semver.

## At a glance

| Metric (size 500 catalog) | Jogak | Storybook 8 | Ratio |
|---|---|---|---|
| Dev cold start | **2.9 s** | 3.6 s | 1.3× |
| Build time | **4.1 s** | 7.6 s | **1.8×** |
| Bundle (gzip) | **156 KB** | 1.09 MB | **7.0×** |
| Idle RSS (dev) | **345 MB** | 489 MB | **1.4×** |
| HMR latency (warm) | **199 ms** | — | < 200 ms |

See [Benchmarks](/en/docs/benchmarks) for the full table and reproduction commands.

## 5 frameworks, one viewer

```ts
const meta = {
  title: 'Components/Button',
  component: Button,
  framework: 'vue', // 'react' | 'next' | 'vue' | 'svelte' | 'web-components'
} satisfies JogakMeta
```

The same `*.jogak.{ts,tsx}` convention works across **React · Next.js · Vue 3 · Svelte 5 · Web Components**. User components mount in your own Vite scope (iframe isolation) — your Tailwind, design system, and bundler stay yours.

## Quick start

```bash
pnpm add -D @jogak/cli @jogak/core

# write *.jogak.tsx next to your components, then:
npx jogak dev          # http://localhost:5173
npx jogak build        # static output → ./jogak-static
```

→ Continue to the [Quick start guide](/en/docs).

## Why "Jogak"?

> "조각들이 모여 하나의 조각을 만든다" — _Small pieces come together to form one whole._
>
> Storybook is heavy and error-prone for many teams. Jogak shares the same intent — a place to **gather component pieces** — but stays small enough to ship as a drop-in.

Try it. Send feedback. The alpha exists to harden the contract before `beta`.
