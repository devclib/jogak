# Roadmap

Jogak is now **`1.0.0`** — stable release. Public API is frozen; future changes follow semver.

## Where we are (July 2026)

- **1.0.0** released — 5 frameworks (React, Next.js, Vue 3, Svelte 5, Web Components), iframe isolation, props auto-extraction (ts-morph), host embedding, 5 starter examples
- CI matrix: `vue`, `svelte`, `react` iframe mount + `next` separate job (long-timeout for cold compile)
- Public API frozen (no breaking changes) — additive features only until 2.0

## Path from alpha to 1.0

| Phase | Milestone |
|---|---|
| `0.1.0-alpha.x` | Engine + adapters + host embedding + 5 examples |
| `1.0.0-beta.0` | alpha → beta promotion (5/5 gate passed) |
| `1.0.0-beta.1 – beta.7` | Stability + convenience fixes: Next 16 turbopack scaffold, chrome scope null guard, iframe height auto-sync, A11y addon (axe-core), Migration/Comparison docs, empty-state UX, vite optimizeDeps auto-config, `jogak.config.ts` auto-restart, Next spawn resolve, parser skip warning |
| **`1.0.0`** | **Stable release** — API frozen |

## Post-1.0 (planned)

- **Themes addon** — toolbar theme switcher (parked in Comparison matrix)
- **MDX docs** — richer per-story documentation (parked)
- **Web Components smoke automation** — currently VR-covered only
- **Full parser skip reason** — extract-core surface breakdown (defaultObj vs title vs framework)
- **Interactions / test runner** — not planned for 1.x (out of scope by design)
