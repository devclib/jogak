# Web Components

Vanilla JS / Custom Elements와 **Shadow DOM 격리**. Preact 런타임만 ship됩니다 (~3 KB) — Web Components 경로에서는 React가 절대 로드되지 않습니다.

## 셋업

```bash
pnpm add -D @jogak/cli @jogak/core
pnpm add preact
pnpm add -D @preact/preset-vite
```

```ts filename="jogak.config.ts"
import { defineJogakConfig } from '@jogak/core'

export default defineJogakConfig({
  framework: 'web-components',
})
```

```ts filename="vite.config.ts"
import { defineConfig } from 'vite'
import preact from '@preact/preset-vite'

export default defineConfig({
  plugins: [preact()],
})
```

```json filename="tsconfig.json"
{
  "compilerOptions": {
    "jsx": "react-jsx",
    "jsxImportSource": "preact"
  }
}
```

## 컴포넌트 정의

Preact JSX (또는 framework-agnostic class component):

```tsx filename="src/components/Pill.tsx"
export type Tone = 'info' | 'success' | 'warning' | 'danger'

export function Pill({ label, tone = 'info' }: { label: string; tone?: Tone }) {
  return <span data-tone={tone}>{label}</span>
}
```

```tsx filename="src/components/Pill.jogak.tsx"
import type { JogakMeta, Jogak } from '@jogak/core'
import { Pill } from './Pill'

const meta = {
  title: 'WC/Pill',
  component: Pill,
  framework: 'web-components',
  argTypes: {
    label: { control: 'text' },
    tone: { control: 'select', options: ['info', 'success', 'warning', 'danger'] },
  },
} satisfies JogakMeta

export default meta

export const Info: Jogak = { name: 'Info', args: { label: 'New', tone: 'info' } }
```

## 어떤 호스트 페이지에서든 실제 Custom Element로 사용

같은 registry로 컴포넌트를 Custom Element로 등록하면, 외부 호스트 페이지(Astro, Hugo, 순수 HTML 등)가 framework 런타임 없이 사용할 수 있습니다:

```ts
import { defineJogakElement } from '@jogak/core/renderers/web-components'
import { entries } from '../.jogak/registry'

for (const entry of entries) {
  const last = entry.title.split('/').pop()!.toLowerCase()
  defineJogakElement(`jogak-${last}`, entry)
}
```

```html
<jogak-pill label="Hello" tone="info"></jogak-pill>
```

Shadow DOM이 호스트 페이지의 스타일을 막아주고, 호스트의 `:root` 디자인 토큰은 여전히 cascade되어 들어옵니다.
