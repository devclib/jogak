# 로드맵

Jogak은 이제 **`1.0.0`** — 정식 릴리스입니다. Public API가 고정됐고, 이후 변경은 semver를 따릅니다.

## 현재 위치 (2026년 7월)

- **1.0.0** 릴리스 — 5개 framework (React, Next.js, Vue 3, Svelte 5, Web Components), iframe 격리, props 자동 추출(ts-morph), 호스트 임베딩, 5개 starter 예시
- CI matrix: `vue`, `svelte`, `react` iframe mount + `next` 별도 job (cold compile 대응 long-timeout)
- Public API 고정 (breaking change 없음) — 2.0 전까지 additive 기능만 추가

## alpha → 1.0 여정

| Phase | Milestone |
|---|---|
| `0.1.0-alpha.x` | 엔진 + 어댑터 + 호스트 임베딩 + 5개 예시 |
| `1.0.0-beta.0` | alpha → beta 승격 (5/5 게이트 통과) |
| `1.0.0-beta.1 – beta.7` | 안정성/편의성 fix: Next 16 turbopack scaffold, chrome scope null guard, iframe height auto-sync, A11y addon (axe-core), Migration/Comparison 문서, empty-state UX, vite optimizeDeps 자동 구성, `jogak.config.ts` auto-restart, Next spawn resolve, parser skip warning |
| **`1.0.0`** | **정식 릴리스** — API 고정 |

## Post-1.0 (계획)

- **Themes addon** — 툴바 theme switcher (Comparison 매트릭스에 planned로 표시)
- **MDX docs** — 풍부한 story별 문서화 (planned)
- **Web Components smoke 자동화** — 현재 VR만 커버
- **Full parser skip reason** — extract-core 표면 세분화 (defaultObj vs title vs framework)
- **Interactions / test runner** — 1.x에서 미계획 (설계상 out of scope)
