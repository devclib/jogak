# Quick start

## 요구사항

- **Node** 20.18+ (또는 22+, 24+) — `fetch`, `AbortSignal.timeout`, `node:test` API 사용
- **TypeScript** 5.5+ — Props 자동 추출에 필요. `tsconfig.json`이 없으면 추출이 건너뛰어지고 `meta.argTypes`로 수동 정의해야 함
- **Vite** 6.x — 호스트 임베딩 시 peer dependency

프레임워크별 peer (택 1):
- **React** 19.x · **Next.js** 14/15 · **Vue** 3.4+ · **Svelte** 5+ · **Web Components** (Preact 런타임 번들 포함, ~3 KB)

## 설치

```bash
pnpm add -D @jogak/cli @jogak/core
```

React라면 이게 전부입니다. 다른 프레임워크라면 framework peer + Vite plugin을 추가하세요 ([Frameworks](/docs/frameworks/react) 섹션 참고).

## 컴포넌트 정의

```tsx filename="src/components/Button.jogak.tsx"
import type { JogakMeta, Jogak } from '@jogak/core'
import { Button } from './Button'

const meta = {
  title: 'Components/Button',
  component: Button,
} satisfies JogakMeta

export default meta

export const Primary: Jogak = {
  name: 'Primary',
  args: { label: 'Click me', variant: 'primary' },
}

export const Disabled: Jogak = {
  name: 'Disabled',
  args: { label: 'Click me', variant: 'primary', disabled: true },
}
```

`Button`의 props 타입(`variant: 'primary' | 'secondary'`, `disabled?: boolean`, `onClick?: (e) => void`)이 자동으로 select / checkbox / Action 컨트롤로 추출됩니다.

## 실행

```bash
npx jogak dev          # http://localhost:5173
npx jogak build        # static 산출물 → ./jogak-static
```

빌드 결과는 `index.html` + 단일 JS chunk입니다. GitHub Pages, Vercel, Netlify, S3 — 어디든 그대로 올리면 됩니다. base path는 `--base /repo-name/`로 조정.

## 얻는 것

- **사이드바** — `meta.title` 기준 자동 트리 (`Components/Button` → `Components` 그룹)
- **Preview** — 사용자 자신의 Vite scope iframe에서 마운트 (Tailwind / 디자인 시스템 그대로 적용)
- **Controls** — TS 타입에서 생성된 Props 에디터 (text · number · boolean · select)
- **Actions** — 함수 prop 자동 spy + 호출 로그
- **소스 뷰** — 현재 args 기준의 사용 코드 자동 생성
- **URL 딥 링크** — `?entry=Components/Button&jogak=Primary`

## 설정 (선택)

```ts filename="jogak.config.ts"
import { defineJogakConfig } from '@jogak/core'

export default defineJogakConfig({
  globalCss: 'src/styles/globals.css',  // 또는 `true`로 자동 감지
  framework: 'react',                    // 'react' | 'next' | 'vue' | 'svelte' | 'web-components'
  previewIsolation: 'iframe',            // default 'iframe' — 권장
})
```

→ 전체 옵션은 [Configuration](/docs/configuration)에 있습니다 (현재 영문).
