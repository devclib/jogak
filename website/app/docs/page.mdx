# Quick start

## Requirements

- **Node** 20.18+ (or 22+, 24+) — uses `fetch`, `AbortSignal.timeout`, `node:test` parity APIs
- **TypeScript** 5.5+ — required for props auto-extraction (without `tsconfig.json`, extraction is skipped and you fall back to manual `meta.argTypes`)
- **Vite** 6.x — peer dependency when embedding

Per-framework peers (any one):
- **React** 19.x · **Next.js** 14/15 · **Vue** 3.4+ · **Svelte** 5+ · **Web Components** (Preact runtime bundled, ~3 KB)

## Install

```bash
pnpm add -D @jogak/cli @jogak/core
```

That's all you need for React. For other frameworks, add the framework peer + its Vite plugin (see the [Frameworks](/docs/frameworks/react) section).

## Define a component

```tsx filename="src/components/Button.jogak.tsx"
import type { JogakMeta, Jogak } from '@jogak/core'
import { Button } from './Button'

const meta = {
  title: 'Components/Button',
  component: Button,
} satisfies JogakMeta

export default meta

export const Primary: Jogak = {
  name: 'Primary',
  args: { label: 'Click me', variant: 'primary' },
}

export const Disabled: Jogak = {
  name: 'Disabled',
  args: { label: 'Click me', variant: 'primary', disabled: true },
}
```

`Button`'s prop types — `variant: 'primary' | 'secondary'`, `disabled?: boolean`, `onClick?: (e) => void` — are auto-extracted into select / checkbox / Action controls.

## Run

```bash
npx jogak dev          # http://localhost:5173
npx jogak build        # static output → ./jogak-static
```

The build output is `index.html` + a single JS chunk. Upload it as-is to GitHub Pages, Vercel, Netlify, S3 — anywhere. Adjust the base path with `--base /repo-name/`.

## What you get

- **Sidebar** — auto tree by `meta.title` (`Components/Button` → `Components` group)
- **Preview** — your component mounted in an iframe scoped to your own Vite (your Tailwind / design system applies)
- **Controls** — Props editor generated from your TS types (text · number · boolean · select)
- **Actions** — function props auto-spied; call logs in the bottom panel
- **Source view** — usage snippet generated from current args
- **URL deep linking** — `?entry=Components/Button&jogak=Primary`

## Configure (optional)

```ts filename="jogak.config.ts"
import { defineJogakConfig } from '@jogak/core'

export default defineJogakConfig({
  globalCss: 'src/styles/globals.css',  // or `true` for auto-detect
  framework: 'react',                    // 'react' | 'next' | 'vue' | 'svelte' | 'web-components'
  previewIsolation: 'iframe',            // default 'iframe' — recommended
})
```

→ Full options in [Configuration](/docs/configuration).

## Examples

Five ready-to-run starters live in [`examples/`](https://github.com/devclib/jogak/tree/main/examples) — each is a self-contained npm project. Copy the folder, `pnpm install`, `pnpm jogak:dev`.

- [`react-vite/`](https://github.com/devclib/jogak/tree/main/examples/react-vite) — React 19 + Vite 6 + Tailwind v4
- [`nextjs/`](https://github.com/devclib/jogak/tree/main/examples/nextjs) — Next.js 15 App Router (Server + Client)
- [`vue/`](https://github.com/devclib/jogak/tree/main/examples/vue) — Vue 3 SFC
- [`svelte/`](https://github.com/devclib/jogak/tree/main/examples/svelte) — Svelte 5 runes
- [`web-components/`](https://github.com/devclib/jogak/tree/main/examples/web-components) — Preact + Custom Elements
